# KOReader contrib plugins
This is a collection of plugins designed for KOReader

These plugins are in general less commonly used and not part of core logic KOReader would like to maintain. They may stop working after updating KOReader.

To users:
* If an application stops working after a certain KOReader version, please kindly create an issue upstream so the owner can fix or remove it.

To developers:
* This is not a dump station of work-in-progress or deprecated features, please ensure it works when merging.
* Each application should be a git submodule pointing to an upstream repo
* Add a README.md / README file to explain the functionality of the application; if it works with certain devices, add a compatibility section in the README or a COMPATIBILITY file.
